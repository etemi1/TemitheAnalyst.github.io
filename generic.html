My Journey

My name is Temi, although everyone calls me Banty. I graduated with a Bachelors in Mathematics and completed my NYSC in 2022. I got interested in data analytics in 2020 when my friends introduced me to the Google Data Analytics Scholarship, i applied and got in, that started my Data Analyst Journey but that's not where it all began.

In my 3rd year in School i started my programming career as a self taught HTML and CSS website developer but it didn't get me so interested as data did, so i stopped and continued my search looking for where my tech passion was in. Oh Well, i did this website as a result of that knowledge so no regrets. In 2019, I started learning how to use historical financial data charts to predict price movements, that initialized my interest for patterns and trends. when i leanrt that Data Analytics was all about using patterns to investigate and gain insights and then use that insights to make better decisions, i finally found my forte, i completed my course in 2021 and started doing my own projects in 2022. I figured i was passionate about helping businesses make driven decisions and help them grow using the insights i get from by building interactive dashboards and Reports. At the end, i have always used data without knowing i did.
